
# 🏘️ Society Resident Management System

A C++ Object-Oriented Programming (OOP) project designed to simplify residential society management. The system provides a centralized platform for managing residents, utility bills, complaints, grocery shopping, hospital appointments, and community events using file handling for data storage.

## ✨ Features

- 👤 Resident Registration & Login
- 📝 Complaint Management
- 💡 Electricity, Gas & Security Bill Payment
- 🛒 Grocery Store with Cart & Invoice
- 🏥 Hospital Appointment & Emergency Booking
- 🎉 Event Booking & Management

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- STL (Vectors, Strings)
- Exception Handling

## 📚 OOP Concepts

- Classes & Objects
- Encapsulation
- Inheritance
- Polymorphism
- Composition
- Virtual Inheritance
- Dynamic Memory Allocation
